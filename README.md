# Task 3: Advanced Styling and JavaScript

**Internship:** Web Development Internship @ ApexPlanet Software Pvt Ltd
**Timeline:** 9 Days
**Task:** 3

## 📌 Objective

Gain advanced skills in CSS and JavaScript for building interactive, responsive websites.

## 🛠️ Steps / Features

### 1. Apply Responsive Design Using Media Queries

- **Objective:** Ensure web pages look great on mobile, tablet, and desktop.
- **Actions:** Used CSS media queries to adjust layouts, font sizes, and other elements across different screen sizes.

### 2. Build an Interactive Quiz or Image Carousel Using JavaScript

- **Objective:** Build more complex JavaScript projects.
- **Actions:** Created an interactive quiz with multiple questions / an image carousel that rotates images based on user interaction.

### 3. Fetch Data from an API Using JavaScript

- **Objective:** Learn how to work with external APIs to fetch real-time data.
- **Actions:** Fetched data from a public API (e.g., weather or joke API) and displayed it dynamically on the webpage.

## 💻 Tech Stack

- HTML5
- CSS3 (Media Queries)
- JavaScript (ES6+)
- Public API (Fetch API)
